## 推奨仕様
- サイズ: 16:9 (1920×1080px)
- 形式: MP4
- コーデック: H.264
- フレームレート: 24-30fps
- ファイルサイズ: 10MB以下推奨
- 長さ: 3-10秒程度
- 音声: なし（ミュート推奨）

## 動作仕様
- ホバー時に自動再生
- ホバー終了時に自動停止
- ループ再生
- ミュート状態で再生（ブラウザの自動再生制限対応）
